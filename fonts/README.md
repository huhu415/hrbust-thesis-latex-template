# 字体目录

本模版正文用**宋体**、标题用**黑体**。这两个字体是 Windows 系统自带的版权字体，
**不能随开源仓库分发**，因此需要你自行放入本目录。

## 需要放入的文件

| 文件名            | 用途      | 来源                                                        |
| ----------------- | --------- | ----------------------------------------------------------- |
| `simsun-win.ttc`  | 宋体（正文） | Windows 系统 `C:\Windows\Fonts\simsun.ttc`，复制后重命名即可 |
| `SimHei.ttf`      | 黑体（标题） | Windows 系统 `C:\Windows\Fonts\simhei.ttf`                  |

> 仿宋、楷体使用 TeX Live 自带的开源 **Fandol** 字体，无需另装。

放好后目录结构应为：

```
fonts/
├── README.md
├── simsun-win.ttc   ← 你自己放入
└── SimHei.ttf       ← 你自己放入
```

## 没有 Windows 字体怎么办？

打开 `main.tex`，把「中文字体」一段注释掉，启用其后的 **Fandol 兜底方案**
（已写好，去掉注释即可）。Fandol 随 TeX Live / Overleaf 提供，可直接编译，
只是与宋体/黑体的观感略有差异。提交学校前建议仍换回宋体/黑体以符合排版要求。
